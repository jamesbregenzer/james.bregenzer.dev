project/radar

WP Core Radar.

A contribution intelligence tool for finding, reviewing, and tracking WordPress Core tickets that are likely to be useful contribution opportunities.

Purpose

Reduce the time spent searching Trac manually by scoring tickets using signals like patch status, testing needs, freshness, ownership, and complexity.

Current state

Live as an independent Cloudflare Worker application with a dashboard, contributions page, and password-protected admin console.

open app →

Stack

Cloudflare Workers, Cloudflare Pages, GitHub-hosted JSON data, WordPress Core Trac signal collection, and static dashboard generation.
