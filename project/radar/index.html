Featured project

WP Core Radar

A contribution workflow tool that scores WordPress Core tickets and helps identify fast, legitimate opportunities to test, review, comment, or watch.

Open live appView contributionsGitHub repo

What it does

Turns Core ticket noise into a prioritized review queue.

Radar collects candidate tickets, scores them using weighted contribution signals, and provides a protected review console for triage notes and decisions.

Why it exists

Spend less time searching. Spend more time contributing.

Good contribution opportunities are often buried in tickets, keywords, comments, and timing signals. Radar brings the strongest candidates to the top.

Input

WordPress ticket signals
