WP Core Radar

A contribution workflow tool that scores WordPress Core tickets and helps identify fast, legitimate opportunities to test, review, comment, or watch.

Open live appView contributionsGitHub repo

What it does

Radar collects candidate WordPress Core tickets, scores them using weighted contribution signals, and provides a protected review console for triage notes and decisions.

Why it exists

Finding good Core contribution opportunities takes time. Radar reduces the search surface so more time can be spent testing, reviewing, and contributing.
